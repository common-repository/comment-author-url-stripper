Removes the comment author URL if it contains any of the specified words.

== Description ==

When writing comments, a lot of people type in google.com, yahoo.com, hotmail.com, and various other web sites which don't belong to them. This plugin strips out the comment author URL whenever it contains anything from a set of words that you specify in the Options screen. The URL will never make it to the database.

== Installation ==

1. Upload `simple-post.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin options at Options->CAU Stripper in the WordPress admin
